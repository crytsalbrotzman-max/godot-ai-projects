# Godot Game Development Portfolio

This repository contains example work created using the Godot Engine.  
The projects demonstrate gameplay scripting, AI behavior systems, physics interactions, and UI development using GDScript.

These prototypes were built to explore game mechanics, asset interaction, and scalable scene systems for modern game development pipelines.


## Project 1 – Physics Interaction Prototype

A physics-based interaction system built in Godot.

Features:
- Rigid body physics
- Collision detection
- Object interaction mechanics
- Environmental triggers
- Real-time physics responses

This prototype demonstrates how game assets interact within a physics-driven environment.


## Project 2 – AI NPC Behavior System

An NPC behavior system created using a simple state machine architecture.

Features:
- Patrol routes
- Player detection
- Chase and return behavior
- Navigation mesh movement
- Modular AI scripting

The system allows NPCs to dynamically react to player presence and environmental triggers.


## Project 3 – UI and Scene Management Demo

A modular UI and scene transition system built with Godot.

Features:
- Dynamic menus
- HUD interface components
- Scene transitions
- Event-driven UI interactions
- Responsive layout system

This project focuses on scalable interface design and scene organization.


## Technologies Used

- Godot Engine
- GDScript
- Scene-based architecture
- Physics and collision systems
- Navigation systems
